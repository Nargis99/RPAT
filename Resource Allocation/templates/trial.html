Summary

All the resources which are not available during any of the month then it is being listed in this sheet.

There are two columns in each month i.e., P6 hours & Available hours.

  1. P6 Hours is the hours assigned in the new/existing project.
  2. Assigned hours is the difference between the UT Available hours & RFT Hours.

For Unique Resources, hours from P6 input file are being summed to the RFT hours and then it is compared with the available utilization hours.

Scenario 1: New Project

  1. Formula, A = UT Hours -(RFT Hours + P6 Hours)

Scenario 2: Existing Project

  1. Formula, A = UT Hours -(RFT Hours + P6 Hours)
* RFT Hours (It is the sum of the hours after removing allocated hours for the project from RFT.)

Interpretation:

  1. If A < 0, then the resource is not available for the month.
  2. If A > 0, then the resource is available for the month.
